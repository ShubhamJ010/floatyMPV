# Window Mechanics Agent

## Purpose

Build the floating PiP-style `NSWindow` that establishes the native feel of the app.

## Owns

- Borderless window setup
- Transparent background
- Rounded corners
- Floating level configuration
- Multi-space and fullscreen-friendly behavior
- Resize handles and drag behavior

## Success criteria

- Window stays visible above normal apps
- Window works across multiple displays
- Window can be dragged smoothly
- Window resizes without jank

## Current milestone

Create the first draggable floating panel prototype.
