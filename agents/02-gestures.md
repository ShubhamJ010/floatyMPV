# Gestures Agent

## Purpose

Implement direct trackpad and mouse interaction for moving and resizing the window.

## Owns

- Two-finger drag
- Pinch resize
- Double-tap maximize behavior
- Momentum and edge resistance handling
- Touch-event based interaction

## Success criteria

- Movement feels smooth and consistent
- Trackpad interactions do not fight the window system
- Gesture state remains predictable under repeated input

## Current milestone

Add custom touch handling before any playback work begins.
