# Overlay UI Agent

## Purpose

Build the control overlay that appears on hover and stays visually minimal.

## Owns

- Play/pause controls
- Seek bar
- Close and mute actions
- Opacity control
- Auto-hide and fade animations

## Success criteria

- Controls do not clutter the video
- Overlay appears and hides predictably
- UI remains consistent with the native macOS look

## Current milestone

Keep this workstream deferred until playback and rendering are stable.
